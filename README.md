Phát triển dự án Web quản lý lực vit (Torque Management System)
Bảng lưu dữ liệu sản xuất tự động từ máy bắn vít gồm các cột:MachineName,Model,Date,SN,Position,torque,Circle,TimeUpload,Result,TimeScrewPosition
CSDL(Database) Lưu dữ liệu lực vít từ máy SQL Sever
Backend (Api)  Xử lý truy vấn lọc dữ liệu,gửi dữ liệu lên web(NodeJs)
Frontend(webUI) Hiển thị dữ liệu , tìm kiếm ,Lọc ...(React)
